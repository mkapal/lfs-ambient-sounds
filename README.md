# LFS Ambient Sounds

## Requirements

- [Node.js](https://nodejs.org/) 18

## Installation

```shell
pnpm install
```

## Development build

```shell
pnpm dev
```

The app connects to `127.0.0.1:29999` by default.

## Production build

```shell
pnpm build
```

## Development

### Custom configuration

Copy `config.toml` to `config.local.toml`, it will take precedence.
